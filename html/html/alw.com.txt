DISSERTATION RESEARCH SUMMARY

    The "chop" code generator uses automatically-generated hard-coded decision
    trees to quickly rewrite a DAG of intermediate code into one of assembler code.
    The decision trees are generated from a database of rewriting rules.  The rules
    are discovered while generating code for a test suite of C code, using symbolic
    descriptions of the semantics of the intermediate and assembler codes.  The final
    hard-coded decision tree is approximately 20 times faster than the equivalent
    component of the Portable C Compiler.
